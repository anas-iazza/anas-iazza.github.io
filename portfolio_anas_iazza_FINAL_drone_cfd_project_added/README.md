- Texte professionnel dans la section À propos.
- Remplacement de "Aéronautique & Simulation" par "Conception & Simulation".
- Ajout d’une section Skills avec des logos réels chargés depuis des sources externes.

> Note : Les logos de la section Skills sont chargés via Internet. Si vous ouvrez le portfolio sans connexion, les noms des outils restent affichés comme fallback.
